# Future-State Journey Map

> How each persona discovers, evaluates, and purchases products using Chartedly — with every current-state friction point addressed.

---

## Design Principles Applied

Before mapping each persona, these are the Chartedly UX principles that eliminate current-state friction:

1. **Netflix-style visual browsing** — Products displayed as rich visual cards in a horizontal-scroll grid, not text-heavy lists
2. **Bilingual by default** — All content available in English and Japanese with instant toggle; no separate "English version"
3. **Trust transparency** — Every ranking shows its methodology, data sources, and recency. No hidden sponsorships
4. **Decision speed** — Every category page has a clear "#1 Pick" with a 3-line rationale visible without scrolling
5. **Cross-category coverage** — Beauty, electronics, home goods, kitchen, stationery, food — all in one platform
6. **Tourist mode** — Dedicated browsing experience for visitors with price benchmarks, tax-free guidance, and store locator
