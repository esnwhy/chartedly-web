## Guidelines

- **Meta titles:** 50-60 characters. Include primary keyword near the front. End with " | Chartedly" brand suffix.
- **Meta descriptions:** 120-155 characters. Include primary keyword, a benefit or differentiator, and a soft CTA.
- Character counts below are for the content only (excluding HTML tags).
