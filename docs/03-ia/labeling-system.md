# Chartedly — Labeling System

**Document version:** 1.0
**Date:** 2026-03-24
**Author:** SAGE (Information Architecture)
**Derived from:** Phase 02 personas, navigation taxonomy, bilingual content architecture

---

## Design Principles for Labels

1. **Natural in both languages.** Every label is written to feel native in EN and JP independently — not translated. Japanese labels use natural phrasing a 22-year-old Yuki or 40-year-old Aiko would expect. English labels use vocabulary a 28-year-old Emma or 35-year-old Marco would recognize instantly.

2. **Short and scannable.** Labels serve mobile-first layouts. Maximum 2 words in English, 6 characters in Japanese where possible for navigation items.

3. **Action-oriented where applicable.** Buttons say what they do ("Buy," "Compare"), not what they are.

4. **No jargon.** No "SKU," no "taxonomy," no "aggregate." Users see "products," "categories," "scores."

## Naming Convention Rules

| Rule | EN Example | JP Example |
|------|-----------|-----------|
| Navigation: nouns or short verbs | Home, Explore, Rankings | ホーム, 探す, ランキング |
| Buttons: verb-first | Buy on Amazon, Compare, Share | Amazonで購入, 比較する, シェア |
| Badges: adjective + noun | #1 Pick, Best Value | イチ推し, コスパ最強 |
| Sections: noun phrase | Key Specs, Full Review | 基本スペック, 詳細レビュー |
| Errors: plain language, no codes | Page not found | ページが見つかりません |
| Filters: value labels, no verbs | Under ¥1,000, Newest | ¥1,000以下, 新しい順 |
| Empty states: problem + suggestion | No results. Try removing a filter. | 結果なし。条件を変更してみてください。 |
